CrcSpin result matrix map

Below is pseudo code describing the expressions used to obtain study_result column values in CrcSpinModel and DukesCrcSpinModel.

The expressions are applied over all persons, then colons (one per person), then adenomas (many per colon) in the study. The object nesting person.colon.adenoma is implied and is not written explicitly for brevity.
